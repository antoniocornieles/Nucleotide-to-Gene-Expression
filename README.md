# 🧬 VariantAnalyzer

**Advanced Analytical Tool** — Detect dominant and recessive alleles from DNA sequences.

A bioinformatics portfolio project that analyzes FASTA sequences and predicts Mendelian traits with probability scoring.

![Project Banner](cover.png)

## ✨ Features

- Upload DNA sequences in **FASTA format**
- Analytical detection of dominant and recessive variants
- Probability scoring with **margin of error**
- Clear genotype classification (Homozygous Ref / Heterozygous / Homozygous Alt)
- Interactive visualizations (phenotype distribution)
- Exportable CSV reports
- Educational trait explanations

## Supported Traits

- PTC Tasting (TAS2R38)
- Lactose Persistence (MCM6)
- Earwax Type (ABCC11)
- Red Hair / Fair Skin (MC1R)
- Cystic Fibrosis Carrier (CFTR)
- Dimples, Tongue Rolling, Hitchhiker's Thumb, Widow's Peak, and more
